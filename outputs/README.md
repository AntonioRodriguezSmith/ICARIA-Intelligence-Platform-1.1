# outputs

En esta carpeta se guardan los **resultados automáticos y salidas temporales** generadas por las ejecuciones de los scripts, modelos y procesos de validación.

## Ejemplos de contenido

- Logs de ejecución y reportes intermedios.
- Archivos de salida generados en pruebas o desarrollo.
- Exportaciones previas a revisión o validación final.

> **Importante:** Limpiar periódicamente esta carpeta para evitar acumulación de archivos innecesarios y facilitar la trazabilidad de outputs relevantes.
