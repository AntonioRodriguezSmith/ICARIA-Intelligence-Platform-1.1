# plantillas

Esta carpeta contiene **plantillas base y formularios** que sirven como punto de partida para la generación de nuevos informes, análisis o documentación recurrente dentro del proyecto.

## Ejemplos

- Plantillas de notebooks, informes o presentaciones.
- Plantillas de scripts parametrizables para análisis repetitivos.
- Formatos para reportes regulatorios, internos o para entregables recurrentes.

> **Nota**: Mantener actualizadas las plantillas para reflejar siempre los estándares y requisitos más recientes del proyecto.
